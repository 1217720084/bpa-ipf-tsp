Building the IPF suite of programs.

1. Install the downloaded tar files to a directory /shr:

mkdir /shr

cd /shr

2. Uncompress and untar this file:

gunzip ipf-3.22.tar.gz

tar -xvf ipf-3.22.tar

3. Descend into the ipf-3.22 subdirectories and build IPF in steps:

cd /shr/ipf-3.22/ipf

./build-libipf.bash

cd /shr/ipf-3.22/ipc

./build-libipc.bash

cd /shr/ipf-3.22/gui

./build-libgui.bash

cd /shr/ipf-3.22/exe

make -f bpf.make

make -f netdat.make

make -f ipf_reports.make

make -f linfloanl.make

make -f ipfsrv.make

make -f gui.make

5. Test out one of the programs (batch powerflow).

cd /shr/ipf-3.22/exe

./bpf TESTDC3.PFC
